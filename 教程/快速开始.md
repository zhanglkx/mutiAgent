## 📋 前置要求

```bash

node -v  # v20.0.0 或更高

npm -v
```

如果没有安装，访问 [nodejs.org](https://nodejs.org/) 下载安装。

---

## 🚀 5步快速开始

### Step 1: 创建项目（2分钟）

```bash
# 创建目录
mkdir my-first-agent && cd my-first-agent

# 初始化 Node.js 项目
npm init -y

# 安装依赖
npm install langchain @langchain/openai dotenv
npm install -D typescript tsx @types/node
```

### Step 2: 配置项目（1分钟）

```bash
# 配置 package.json
npm pkg set type="module"
npm pkg set scripts.start="tsx src/index.ts"

# 创建源码目录
mkdir src
```

### Step 3: 配置 API Key（2分钟）

创建 `.env` 文件：

```bash
# 在项目根目录创建 .env
cat > .env << 'EOF'
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxx
EOF
```

**获取 API Key**：
- OpenAI: 访问 [platform.openai.com](https://platform.openai.com/api-keys)
- Claude: 访问 [console.anthropic.com](https://console.anthropic.com/)

---

### Step 4: 编写第一个程序（3分钟）

创建 `src/index.ts`：

```typescript
import { ChatOpenAI } from '@langchain/openai';
import 'dotenv/config';

// 1. 初始化 AI 模型
const llm = new ChatOpenAI({
  modelName: 'gpt-4o-mini',
  temperature: 0.7,
});

// 2. 发送消息
async function main() {
  console.log('🤖 AI Agent 启动中...\n');

  const response = await llm.invoke('用一句话介绍什么是 AI Agent');

  console.log('💬 AI 回复:');
  console.log(response.content);
}

main();
```

### Step 5: 运行程序（1分钟）

```bash
npm start
```

**预期输出**：

```
🤖 AI Agent 启动中...

💬 AI 回复:
AI Agent 是一种能够自主感知环境、做出决策并执行任务的智能系统。
```

## 💡 快速示例

### 示例 1：带工具的 Agent（5分钟）

```typescript
import { ChatOpenAI } from '@langchain/openai';
import { tool } from '@langchain/core/tools';
import { z } from 'zod';
import { createReactAgent } from '@langchain/langgraph/prebuilt';
import { HumanMessage } from '@langchain/core/messages';

// 定义工具
const weatherTool = tool(
  async ({ city }) => {
    const weather = {
      '北京': '25°C，晴天',
      '上海': '28°C，多云',
    };
    return weather[city] || '暂无数据';
  },
  {
    name: 'get_weather',
    description: '获取城市天气',
    schema: z.object({
      city: z.string().describe('城市名称'),
    }),
  }
);

// 创建 Agent
const agent = createReactAgent({
  llm: new ChatOpenAI({ modelName: 'gpt-4o-mini' }),
  tools: [weatherTool],
});

// 使用
const result = await agent.invoke({
  messages: [new HumanMessage('北京天气怎么样？')],
});

console.log(result.messages[result.messages.length - 1].content);
```

### 示例 2：流式输出（5分钟）

```typescript
import { ChatOpenAI } from '@langchain/openai';
import { HumanMessage } from '@langchain/core/messages';

const llm = new ChatOpenAI({
  modelName: 'gpt-4o-mini',
  streaming: true,
});

// 流式输出
const stream = await llm.stream([
  new HumanMessage('用 50 字介绍 TypeScript'),
]);

for await (const chunk of stream) {
  process.stdout.write(chunk.content);
}
```

## 🐛 常见问题

### 问题 1: API Key 错误

```
Error: Incorrect API key provided
```

**解决**：
1. 检查 `.env` 文件中的 API Key 是否正确
2. 确保 `.env` 在项目根目录
3. 重启程序

---

### 问题 2: 模块找不到

```
Error: Cannot find package 'langchain'
```

**解决**：
```bash
# 重新安装依赖
rm -rf node_modules package-lock.json
npm install
```

---

### 问题 3: TypeScript 错误

```
Error: Cannot use import statement outside a module
```

**解决**：
在 `package.json` 中添加：
```json
{
  "type": "module"
}
```
