# 01 - 流式输出

> **核心理念**：像 ChatGPT 一样逐字输出，提升用户体验

## 🎯 学习目标

- 理解流式输出的价值
- 掌握三种流式模式
- 在前端实现逐字显示
- 处理流式错误

---

## 📝 为什么需要流式输出？

### 对比

```typescript
// ❌ 非流式：等待全部完成
用户: "写一篇 1000 字的文章"
[等待 30 秒...]
AI: [一次性返回全部内容]

// ✅ 流式：逐字显示
用户: "写一篇 1000 字的文章"
AI: "在..."
AI: "在人工..."
AI: "在人工智能..."
[持续输出...]
```

**优势**：

- ✅ 即时反馈
- ✅ 更好的用户体验
- ✅ 感知速度更快

## 🛠️ LangGraph 的流式模式

### 1. stream() - 查看每个节点

```typescript
import { createReactAgent } from '@langchain/langgraph/prebuilt';
import { ChatOpenAI } from '@langchain/openai';
import { HumanMessage } from '@langchain/core/messages';

const agent = createReactAgent({
  llm: new ChatOpenAI({ modelName: 'gpt-4o-mini' }),
  tools: [],
});

// 流式输出每个节点的结果
for await (const chunk of agent.stream({
  messages: [new HumanMessage('你好')],
})) {
  console.log('Chunk:', chunk);
}
```

---

### 2. streamEvents() - 查看所有事件

```typescript
// 更细粒度的流式输出
for await (const event of agent.streamEvents(
  { messages: [new HumanMessage('你好')] },
  { version: 'v2' }
)) {
  console.log(`Event: ${event.event} | Name: ${event.name}`);

  if (event.event === 'on_chat_model_stream') {
    // AI 的每个 token
    console.log(event.data.chunk.content);
  }
}
```

### 3. 逐 Token 流式输出

创建 `src/25-streaming.ts`：

```typescript
import { createReactAgent } from '@langchain/langgraph/prebuilt';
import { ChatOpenAI } from '@langchain/openai';
import { HumanMessage } from '@langchain/core/messages';
import 'dotenv/config';

/**
 * 流式输出示例
 * 实现类似 ChatGPT 的逐字效果
 */

const agent = createReactAgent({
  llm: new ChatOpenAI({
    modelName: 'gpt-4o-mini',
    streaming: true, // 👈 开启流式
  }),
  tools: [],
});

async function streamChat(message: string) {
  console.log('👤 用户:', message);
  console.log('🤖 AI: ');

  let fullResponse = '';

  for await (const event of agent.streamEvents(
    { messages: [new HumanMessage(message)] },
    { version: 'v2' }
  )) {
    // 只关注 AI 的输出
    if (event.event === 'on_chat_model_stream') {
      const chunk = event.data?.chunk?.content;

      if (chunk) {
        process.stdout.write(chunk); // 不换行打印
        fullResponse += chunk;
      }
    }
  }

  console.log('\n'); // 换行
  return fullResponse;
}

// 测试
async function main() {
  await streamChat('用 50 字介绍 LangGraph');
}

main();
```

**运行效果**：

```
👤 用户: 用 50 字介绍 LangGraph
🤖 AI: Lang...Graph...是...一个...用于...构建...AI...Agent...的...框架...
```

## 📊 Next.js 集成示例

创建 `app/api/chat/route.ts`：

```typescript
import { createReactAgent } from '@langchain/langgraph/prebuilt';
import { ChatOpenAI } from '@langchain/openai';
import { HumanMessage } from '@langchain/core/messages';

export const runtime = 'edge'; // 使用 Edge Runtime

export async function POST(req: Request) {
  const { message } = await req.json();

  const agent = createReactAgent({
    llm: new ChatOpenAI({
      modelName: 'gpt-4o-mini',
      streaming: true,
    }),
    tools: [],
  });

  // 创建流式响应
  const encoder = new TextEncoder();
  const stream = new ReadableStream({
    async start(controller) {
      try {
        for await (const event of agent.streamEvents(
          { messages: [new HumanMessage(message)] },
          { version: 'v2' }
        )) {
          if (event.event === 'on_chat_model_stream') {
            const chunk = event.data?.chunk?.content;

            if (chunk) {
              controller.enqueue(encoder.encode(chunk));
            }
          }
        }
      } finally {
        controller.close();
      }
    },
  });

  return new Response(stream, {
    headers: {
      'Content-Type': 'text/event-stream',
      'Cache-Control': 'no-cache',
      Connection: 'keep-alive',
    },
  });
}
```

### React 前端组件

```typescript
'use client';

import { useState } from 'react';

export default function ChatPage() {
  const [message, setMessage] = useState('');
  const [response, setResponse] = useState('');
  const [loading, setLoading] = useState(false);

  const handleSend = async () => {
    setLoading(true);
    setResponse('');

    const res = await fetch('/api/chat', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ message }),
    });

    const reader = res.body?.getReader();
    const decoder = new TextDecoder();

    if (reader) {
      while (true) {
        const { done, value } = await reader.read();
        if (done) break;

        const chunk = decoder.decode(value);
        setResponse((prev) => prev + chunk);
      }
    }

    setLoading(false);
  };

  return (
    <div className="p-4">
      <input
        value={message}
        onChange={(e) => setMessage(e.target.value)}
        placeholder="输入消息..."
        className="border p-2 w-full"
      />
      <button
        onClick={handleSend}
        disabled={loading}
        className="mt-2 bg-blue-500 text-white px-4 py-2 rounded"
      >
        {loading ? '发送中...' : '发送'}
      </button>

      <div className="mt-4 whitespace-pre-wrap">{response}</div>
    </div>
  );
}
```

## 💡 常见问题 FAQ

**Q: 所有模型都支持流式吗？**
A: 大部分模型支持，但需要在初始化时设置 `streaming: true`。

**Q: 如何取消流式输出？**
A: 使用 AbortController：

```typescript
const controller = new AbortController();
const stream = agent.stream(input, {
  signal: controller.signal,
});

// 取消
controller.abort();
```

**Q: 流式输出会增加成本吗？**
A: 不会。成本只取决于 token 数量，与是否流式无关。
