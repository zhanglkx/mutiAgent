## 📝 什么是 Function Calling？

### 核心概念

**Function Calling** 让 AI 能够：
1. 理解你定义的函数
2. 决定什么时候调用
3. 提取函数所需的参数
4. 等待你执行后，根据结果继续回答

**前端类比**：

```typescript
// 普通 API 调用
const data = await fetch('/api/weather').then(r => r.json());

// AI Function Calling
// 1. AI 理解用户意图："查天气"
// 2. AI 决定调用：getWeather({ city: "北京" })
// 3. 你执行函数，返回结果
// 4. AI 根据结果回答用户
```

## 🛠️ 基础用法

### Step 1: 定义工具

```typescript
import { tool } from '@langchain/core/tools';
import { z } from 'zod';

/**
 * 定义一个获取天气的工具
 * 前端类比：定义一个 API 接口
 */
const getWeatherTool = tool(
  async ({ city }) => {
    // 实际应该调用天气 API，这里模拟数据
    const weatherData: Record<string, string> = {
      '北京': '25°C，晴天',
      '上海': '28°C，多云',
      '广州': '32°C，雷阵雨',
    };

    return weatherData[city] || '暂无该城市天气数据';
  },
  {
    name: 'get_weather',
    description: '获取指定城市的天气信息',
    schema: z.object({
      city: z.string().describe('城市名称，例如：北京、上海'),
    }),
  }
);
```

### Step 2: 绑定工具到 LLM

```typescript
import { ChatOpenAI } from '@langchain/openai';

const llm = new ChatOpenAI({
  modelName: 'gpt-4o-mini',
  temperature: 0,
});

// 绑定工具（告诉 AI 有哪些函数可用）
const llmWithTools = llm.bindTools([getWeatherTool]);
```

### Step 3: AI 决定是否调用工具

```typescript
// 用户问题
const response = await llmWithTools.invoke('北京今天天气怎么样？');

console.log('AI 的响应：', response);
console.log('工具调用：', response.tool_calls);

// 输出：
// tool_calls: [
//   {
//     name: "get_weather",
//     args: { city: "北京" },
//     id: "call_xxx"
//   }
// ]
```

**重要**：AI 只是**决定**调用工具，但不会真正执行！你需要手动执行。

---

### Step 4: 执行工具并返回结果

```typescript
import { HumanMessage, AIMessage, ToolMessage } from '@langchain/core/messages';

// 1. 用户问题
const messages = [new HumanMessage('北京今天天气怎么样？')];

// 2. AI 决定调用工具
const response1 = await llmWithTools.invoke(messages);
messages.push(response1);  // 保存 AI 的响应

// 3. 手动执行工具
if (response1.tool_calls && response1.tool_calls.length > 0) {
  for (const toolCall of response1.tool_calls) {
    // 执行工具
    const toolResult = await getWeatherTool.invoke(toolCall.args);

    // 添加工具执行结果
    messages.push(
      new ToolMessage({
        content: toolResult,
        tool_call_id: toolCall.id,
      })
    );
  }
}

// 4. AI 根据工具结果回答用户
const response2 = await llmWithTools.invoke(messages);
console.log('最终回答：', response2.content);
// 输出："北京今天天气是 25°C，晴天。"
```

## 🎨 完整示例：智能助手

创建 `src/10-weather-assistant.ts`：

```typescript
import { ChatOpenAI } from '@langchain/openai';
import { tool } from '@langchain/core/tools';
import { z } from 'zod';
import {
  HumanMessage,
  AIMessage,
  ToolMessage,
  BaseMessage,
} from '@langchain/core/messages';
import 'dotenv/config';

/**
 * 智能天气助手
 * 功能：自动判断是否需要查询天气，并返回结果
 */

// 1. 定义工具
const getWeatherTool = tool(
  async ({ city }) => {
    console.log(`📡 正在查询 ${city} 的天气...`);

    // 模拟 API 调用
    await new Promise((resolve) => setTimeout(resolve, 1000));

    const weatherData: Record<string, string> = {
      '北京': '25°C，晴天',
      '上海': '28°C，多云',
      '广州': '32°C，雷阵雨',
      '深圳': '30°C，晴天',
    };

    return weatherData[city] || '暂无该城市天气数据';
  },
  {
    name: 'get_weather',
    description: '获取指定城市的当前天气信息',
    schema: z.object({
      city: z.string().describe('城市名称'),
    }),
  }
);

const getCurrentTimeTool = tool(
  async () => {
    return new Date().toLocaleString('zh-CN');
  },
  {
    name: 'get_current_time',
    description: '获取当前时间',
    schema: z.object({}),
  }
);

// 2. 创建助手类
class WeatherAssistant {
  private llm: ChatOpenAI;
  private tools: any[];

  constructor() {
    this.tools = [getWeatherTool, getCurrentTimeTool];

    this.llm = new ChatOpenAI({
      modelName: 'gpt-4o-mini',
      temperature: 0,
    }).bindTools(this.tools);
  }

  async chat(userInput: string): Promise<string> {
    const messages: BaseMessage[] = [new HumanMessage(userInput)];

    while (true) {
      // AI 响应
      const response = await this.llm.invoke(messages);
      messages.push(response);

      // 检查是否需要调用工具
      if (!response.tool_calls || response.tool_calls.length === 0) {
        // 没有工具调用，直接返回结果
        return response.content as string;
      }

      // 执行所有工具调用
      for (const toolCall of response.tool_calls) {
        console.log(`🔧 调用工具: ${toolCall.name}`);
        console.log(`📥 参数: ${JSON.stringify(toolCall.args)}`);

        // 找到对应的工具并执行
        const tool = this.tools.find((t) => t.name === toolCall.name);
        if (tool) {
          const result = await tool.invoke(toolCall.args);
          console.log(`📤 结果: ${result}\n`);

          // 添加工具执行结果
          messages.push(
            new ToolMessage({
              content: result,
              tool_call_id: toolCall.id!,
            })
          );
        }
      }

      // 继续循环，让 AI 根据工具结果生成最终答案
    }
  }
}

// 3. 测试
async function main() {
  const assistant = new WeatherAssistant();

  console.log('='.repeat(50));
  console.log('测试 1：查询天气');
  console.log('='.repeat(50));
  const answer1 = await assistant.chat('北京今天天气怎么样？');
  console.log('💬 最终回答：', answer1);

  console.log('\n' + '='.repeat(50));
  console.log('测试 2：查询时间');
  console.log('='.repeat(50));
  const answer2 = await assistant.chat('现在几点了？');
  console.log('💬 最终回答：', answer2);

  console.log('\n' + '='.repeat(50));
  console.log('测试 3：普通问答（不需要工具）');
  console.log('='.repeat(50));
  const answer3 = await assistant.chat('什么是 React？');
  console.log('💬 最终回答：', answer3);
}

main();
```

**运行结果：**

```

测试 1：查询天气
==================================================
🔧 调用工具: get_weather
📥 参数: {"city":"北京"}
📡 正在查询 北京 的天气...
📤 结果: 25°C，晴天

💬 最终回答： 北京今天的天气是25°C，晴天。

==================================================

==================================================
🔧 调用工具: get_current_time
📥 参数: {}
📤 结果: 2024/3/16 10:30:00

💬 最终回答： 现在是2024年3月16日 10:30:00。

==================================================
```

## 📊 多工具协作

### 示例：计算器 + 货币转换

```typescript
/**
 * 多个工具协作完成复杂任务
 */

// 工具1：计算器
const calculatorTool = tool(
  async ({ expression }) => {
    try {
      // 注意：实际项目中不要用 eval，这里仅示例
      const result = eval(expression);
      return `${expression} = ${result}`;
    } catch (error) {
      return '计算错误';
    }
  },
  {
    name: 'calculator',
    description: '执行数学计算，支持 +、-、*、/ 等运算',
    schema: z.object({
      expression: z.string().describe('数学表达式，如：2+3*4'),
    }),
  }
);

// 工具2：货币转换
const currencyTool = tool(
  async ({ amount, from, to }) => {
    const rates: Record<string, Record<string, number>> = {
      CNY: { USD: 0.14, EUR: 0.13 },
      USD: { CNY: 7.2, EUR: 0.92 },
      EUR: { CNY: 7.8, USD: 1.09 },
    };

    const rate = rates[from]?.[to] || 1;
    const result = amount * rate;
    return `${amount} ${from} = ${result.toFixed(2)} ${to}`;
  },
  {
    name: 'currency_converter',
    description: '货币转换',
    schema: z.object({
      amount: z.number().describe('金额'),
      from: z.enum(['CNY', 'USD', 'EUR']).describe('源货币'),
      to: z.enum(['CNY', 'USD', 'EUR']).describe('目标货币'),
    }),
  }
);

// 绑定多个工具
const llmWithTools = llm.bindTools([calculatorTool, currencyTool]);

// 测试：复杂任务
const assistant = new WeatherAssistant();
const answer = await assistant.chat('帮我计算 100+200，然后把结果转换成美元');

// AI 会自动：
// 1. 调用 calculator("100+200") → 300
// 2. 调用 currency_converter(300, "CNY", "USD") → 42 USD
// 3. 返回最终答案："100+200=300，转换成美元约为 42 USD"
```

## 🔍 工具定义最佳实践

### 1. 清晰的名称和描述

```typescript
// ❌ 不好：描述模糊
const badTool = tool(
  async ({ x }) => x * 2,
  {
    name: 'func',
    description: 'do something',
    schema: z.object({ x: z.number() }),
  }
);

// ✅ 好：名称和描述清晰
const goodTool = tool(
  async ({ number }) => number * 2,
  {
    name: 'double_number',
    description: '将输入的数字乘以2，返回结果',
    schema: z.object({
      number: z.number().describe('要翻倍的数字'),
    }),
  }
);
```

### 2. 参数添加详细描述

```typescript
const searchTool = tool(
  async ({ query, maxResults }) => {
    // 搜索逻辑...
  },
  {
    name: 'search',
    description: '在数据库中搜索内容',
    schema: z.object({
      query: z.string()
        .describe('搜索关键词，支持中文和英文'),
      maxResults: z.number()
        .default(10)
        .describe('返回结果的最大数量，默认10条'),
    }),
  }
);
```

### 3. 处理错误

```typescript
const apiTool = tool(
  async ({ url }) => {
    try {
      const response = await fetch(url);
      const data = await response.json();
      return JSON.stringify(data);
    } catch (error) {
      // 返回错误信息，让 AI 知道出错了
      return `调用 API 失败: ${error.message}`;
    }
  },
  {
    name: 'call_api',
    description: '调用外部 API',
    schema: z.object({
      url: z.string().url().describe('API 地址'),
    }),
  }
);
```

## 🎯 实战项目：文件管理助手

创建 `src/11-file-assistant.ts`：

```typescript
import { ChatOpenAI } from '@langchain/openai';
import { tool } from '@langchain/core/tools';
import { z } from 'zod';
import * as fs from 'fs/promises';
import * as path from 'path';

/**
 * 文件管理助手
 * 功能：读取、写入、列出文件
 */

// 工具1：列出文件
const listFilesTool = tool(
  async ({ directory }) => {
    try {
      const files = await fs.readdir(directory);
      return `文件列表：\n${files.join('\n')}`;
    } catch (error) {
      return `无法访问目录：${error.message}`;
    }
  },
  {
    name: 'list_files',
    description: '列出指定目录下的所有文件',
    schema: z.object({
      directory: z.string().describe('目录路径'),
    }),
  }
);

// 工具2：读取文件
const readFileTool = tool(
  async ({ filePath }) => {
    try {
      const content = await fs.readFile(filePath, 'utf-8');
      return `文件内容：\n${content}`;
    } catch (error) {
      return `无法读取文件：${error.message}`;
    }
  },
  {
    name: 'read_file',
    description: '读取文件内容',
    schema: z.object({
      filePath: z.string().describe('文件路径'),
    }),
  }
);

// 工具3：写入文件
const writeFileTool = tool(
  async ({ filePath, content }) => {
    try {
      await fs.writeFile(filePath, content, 'utf-8');
      return `文件已写入：${filePath}`;
    } catch (error) {
      return `无法写入文件：${error.message}`;
    }
  },
  {
    name: 'write_file',
    description: '写入内容到文件',
    schema: z.object({
      filePath: z.string().describe('文件路径'),
      content: z.string().describe('要写入的内容'),
    }),
  }
);

  private llm: ChatOpenAI;

  constructor() {
    this.tools = [listFilesTool, readFileTool, writeFileTool];
    this.llm = new ChatOpenAI({
      modelName: 'gpt-4o-mini',
      temperature: 0,
    }).bindTools(this.tools);
  }

  async chat(userInput: string): Promise<string> {
    // ... (与 WeatherAssistant 相同的实现)
  }
}

// 测试
const assistant = new FileAssistant();

await assistant.chat('读取 hello.txt 文件的内容');
await assistant.chat('列出当前目录下的所有文件');
```

## 📚 知识点总结

### ✅ 你已经掌握

1. **Function Calling 核心概念**
   - AI 决定调用哪个函数
   - 提取函数参数
   - 根据执行结果继续对话

2. **工具定义**
   - 使用 `tool()` 定义工具
   - Zod Schema 定义参数
   - 清晰的名称和描述

3. **工具执行流程**
   - 绑定工具：`llm.bindTools()`
   - AI 响应：`response.tool_calls`
   - 手动执行：`tool.invoke()`
   - 返回结果：`ToolMessage`

4. **实战应用**
   - 天气查询
   - 计算器
   - 文件管理

### 🔄 前端类比汇总

| Function Calling 概念 | 前端类比 | 说明 |
|---------------------|---------|------|
| 工具定义 | API 接口定义 | 定义函数签名 |
| `bindTools()` | 注册路由 | 告诉系统有哪些接口 |
| `tool_calls` | API 请求 | AI 发起调用请求 |
| 执行工具 | 处理请求 | 真正执行函数 |
| `ToolMessage` | API 响应 | 返回执行结果 |

## 💡 常见问题 FAQ

**Q: AI 什么时候会调用工具？**
A: 完全由 AI 自己决定。你只需要提供清晰的工具描述，AI 会根据用户问题判断是否需要调用。

**Q: 可以强制 AI 调用某个工具吗？**
A: 可以！使用 `tool_choice` 参数：
```typescript
const llm = new ChatOpenAI({
  modelName: 'gpt-4o-mini',
}).bindTools([tool], {
  tool_choice: 'get_weather',  // 强制调用这个工具
});
```

**Q: 工具执行失败了怎么办？**
A: 在工具中捕获错误，返回错误信息：
```typescript
const tool = tool(async ({ x }) => {
  try {
    return await dangerousOperation(x);
  } catch (error) {
    return `操作失败：${error.message}`;
  }
}, {...});
```

**Q: 可以调用异步 API 吗？**
A: 当然！工具函数支持 `async/await`：
```typescript
const tool = tool(
  async ({ url }) => {
    const response = await fetch(url);
    return await response.text();
  },
  {...}
);
```

**Q: 如何限制工具的调用权限？**
A: 在工具内部添加验证逻辑：
```typescript
const deleteTool = tool(
  async ({ filePath, confirmCode }) => {
    if (confirmCode !== 'DELETE_CONFIRM') {
      return '请提供确认码以执行删除操作';
    }
    // 执行删除...
  },
  {...}
);
```
