## 📝 核心概念

### 1. 什么是大模型 API？

**你已经熟悉的模式：**

```typescript
// 调用普通后端 API
const response = await fetch('/api/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ message: '你好' }),
});
const data = await response.json();
console.log(data.reply); // "你好！有什么可以帮你？"
```

**调用大模型 API（LangChain.js）：**

```typescript
import { ChatOpenAI } from '@langchain/openai';

// 初始化模型（类似创建 axios 实例）
const llm = new ChatOpenAI({
  modelName: 'gpt-4o-mini',
  temperature: 0.7,
});

// 发送请求
const response = await llm.invoke('你好');
console.log(response.content); // "你好！有什么可以帮你？"
```

**核心区别**：

- 普通 API：返回确定的结果（调用 100 次，返回相同）
- 大模型 API：返回概率性结果（每次可能不同）

## 🧠 深入理解

### 1. LangChain.js 的价值

**为什么不直接用 OpenAI SDK？**

```typescript
// ❌ 直接用 OpenAI SDK - 切换模型麻烦
import OpenAI from 'openai';
const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });
const response = await openai.chat.completions.create({
  model: 'gpt-4',
  messages: [{ role: 'user', content: '你好' }],
});

// ✅ 用 LangChain.js - 统一接口，轻松切换
import { ChatOpenAI } from '@langchain/openai';
import { ChatAnthropic } from '@langchain/anthropic';

const llm = new ChatOpenAI({ modelName: 'gpt-4o-mini' });
// 切换到 Claude 只需改一行！
const llm2 = new ChatAnthropic({ modelName: 'claude-3-5-sonnet-20241022' });
```

**类比**：就像 Axios 统一了 `fetch` 和 `XMLHttpRequest`。

---

### 2. 关键参数详解

```typescript
const llm = new ChatOpenAI({
  modelName: 'gpt-4o-mini', // 🎯 模型选择
  temperature: 0.7, // 🎲 随机性控制（重要！）
  maxTokens: 500, // 📏 输出长度限制
  timeout: 30000, // ⏱️ 请求超时（毫秒）
  maxRetries: 2, // 🔄 重试次数
});
```

#### Temperature 参数详解（超级重要！）

```typescript
// temperature = 0（确定性输出）
const deterministicLLM = new ChatOpenAI({ temperature: 0 });
await deterministicLLM.invoke('1+1=?'); // 总是返回 "2"

// temperature = 1（创造性输出）
const creativeLLM = new ChatOpenAI({ temperature: 1 });
await creativeLLM.invoke('写一首诗'); // 每次都不同
```

**使用场景：**

| Temperature | 适用场景     | 示例                         |
| ----------- | ------------ | ---------------------------- |
| 0 - 0.3     | 需要准确答案 | 数据提取、代码生成、数学计算 |
| 0.5 - 0.7   | 日常对话     | 客服机器人、问答系统         |
| 0.8 - 1.0   | 创意任务     | 写作、头脑风暴、艺术创作     |

## 🎨 进阶示例

### 示例 1：多轮对话（❌ 错误示范）

```typescript
const llm = new ChatOpenAI({ modelName: 'gpt-4o-mini' });

// ❌ 错误：大模型没有记忆！
await llm.invoke('我叫小明');
const response = await llm.invoke('我叫什么名字？');
console.log(response.content); // 输出："我不知道您的名字"
```

**为什么？** 每次 `invoke` 都是独立的请求，模型不知道上一轮说了什么！

---

### 示例 2：多轮对话（✅ 正确做法）

```typescript
import { HumanMessage, AIMessage } from '@langchain/core/messages';

const llm = new ChatOpenAI({ modelName: 'gpt-4o-mini' });

// 手动管理对话历史（类似 useState）
const messages = [
  new HumanMessage('我叫小明'),
  new AIMessage('你好，小明！'),
  new HumanMessage('我叫什么名字？'),
];

const response = await llm.invoke(messages);
console.log(response.content); // 输出："你叫小明"
```

**React 类比：**

```tsx
// React 中管理对话历史
const [messages, setMessages] = useState<Message[]>([
  { role: 'user', content: '我叫小明' },
  { role: 'assistant', content: '你好，小明！' },
]);

const addMessage = (msg: Message) => {
  setMessages([...messages, msg]);
};
```

## 📊 完整示例：交互式聊天

创建 `src/02-interactive-chat.ts`：

```typescript
import { ChatOpenAI } from '@langchain/openai';
import { HumanMessage, AIMessage, SystemMessage } from '@langchain/core/messages';
import * as readline from 'readline';
import 'dotenv/config';

/**
 * 交互式聊天程序
 * 功能：命令行版 ChatGPT
 */

// 初始化模型
const llm = new ChatOpenAI({
  modelName: 'gpt-4o-mini',
  temperature: 0.7,
});

// 对话历史（类似 React 的 state）
const conversationHistory = [new SystemMessage('你是一个友好的 AI 助手，用简洁的语言回答问题。')];

// 创建命令行输入接口
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

// 工具函数：获取用户输入
function askQuestion(prompt: string): Promise<string> {
  return new Promise(resolve => {
    rl.question(prompt, answer => {
      resolve(answer);
    });
  });
}

// 主函数
async function main() {
  console.log('🤖 AI 助手已启动！输入 "quit" 退出\n');

  while (true) {
    // 获取用户输入
    const userInput = await askQuestion('👤 你: ');

    if (userInput.toLowerCase() === 'quit') {
      console.log('👋 再见！');
      rl.close();
      break;
    }

    // 添加用户消息到历史
    conversationHistory.push(new HumanMessage(userInput));

    // 调用模型
    const response = await llm.invoke(conversationHistory);

    // 添加 AI 回复到历史
    conversationHistory.push(new AIMessage(response.content as string));

    // 打印回复
    console.log(`\n🤖 AI: ${response.content}\n`);

    // 可选：查看 token 使用情况
    const tokenUsage = response.response_metadata.tokenUsage;
    console.log(`💰 Token 使用: ${tokenUsage.totalTokens}\n`);
  }
}

main();
```

**运行效果：**

```
🤖 AI 助手已启动！输入 "quit" 退出

👤 你: 我叫小明

🤖 AI: 你好，小明！很高兴认识你。

💰 Token 使用: 45

👤 你: 我叫什么名字？

🤖 AI: 你叫小明。

💰 Token 使用: 78

👤 你: quit
👋 再见！
```

### 2. Token 计数器

```typescript
let totalTokens = 0;

async function chatWithCounter(message: string) {
  const response = await llm.invoke(message);
  const tokens = response.response_metadata.tokenUsage.totalTokens;
  totalTokens += tokens;

  console.log(`📊 本次: ${tokens} | 累计: ${totalTokens}`);
  return response.content;
}

await chatWithCounter('你好');
await chatWithCounter('介绍一下 React');
```

### 任务 2：错误处理（10分钟）

添加网络错误、API 限流的处理逻辑：

```typescript
async function chatWithRetry(message: string, maxRetries = 3): Promise<string> {
  for (let i = 0; i < maxRetries; i++) {
    try {
      const response = await llm.invoke(message);
      return response.content as string;
    } catch (error) {
      console.error(`❌ 错误: ${error}`);

      if (i < maxRetries - 1) {
        const waitTime = Math.pow(2, i) * 1000; // 指数退避
        console.log(`⏳ ${waitTime / 1000}秒后重试...`);
        await new Promise(resolve => setTimeout(resolve, waitTime));
      } else {
        throw error;
      }
    }
  }
  throw new Error('Max retries reached');
}
```

## 💡 常见问题 FAQ

**Q: 为什么选择 LangChain.js 而不是直接用 OpenAI SDK？**

A: LangChain 提供了更高级的抽象：

- ✅ 统一的 API（轻松切换模型）
- ✅ 内置记忆管理
- ✅ 工具调用支持
- ✅ 与 LangGraph 无缝集成

**Q: Temperature 设置为多少合适？**

A:

- 需要准确答案：`0 - 0.3`
- 日常对话：`0.7`
- 创意写作：`0.8 - 1.0`

**Q: 如何节省 Token 费用？**

A:

1. 使用更小的模型（`gpt-4o-mini` 而不是 `gpt-4`）
2. 限制 `maxTokens`
3. 精简对话历史（只保留最近 10 轮）
4. 使用缓存（相同问题不重复请求）

**Q: 可以用 Claude 模型吗？**

A: 当然！只需替换：

```typescript
import { ChatAnthropic } from '@langchain/anthropic';

const llm = new ChatAnthropic({
  modelName: 'claude-3-5-sonnet-20241022',
  temperature: 0.7,
});
```

**Q: 如何在 Next.js 中使用？**

A: 第五阶段会详细讲解，简单预览：

```typescript
// app/api/chat/route.ts
import { ChatOpenAI } from '@langchain/openai';

export async function POST(req: Request) {
  const { message } = await req.json();
  const llm = new ChatOpenAI();
  const response = await llm.invoke(message);
  return Response.json({ reply: response.content });
}
```
