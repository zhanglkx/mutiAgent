# 03 - 提示词模板

> **核心理念**：Prompt Template 就像 React 的 JSX 模板，动态插入变量

## 🎯 学习目标

- 掌握提示词模板的使用
- 理解动态变量注入

- 理解 ChatPromptTemplate 和 MessagesPlaceholder

## 📝 为什么需要 Prompt Template？

### 问题：硬编码的 Prompt 难以维护

```typescript
// ❌ 硬编码 - 每次都要修改代码
const response = await llm.invoke(`
你是一个 React 专家。
用户问题：如何使用 useState？
请用简洁的语言回答。
`);

// 如果有 100 个不同的问题，就要写 100 次？
```

### 解决方案：使用模板

```typescript
// ✅ 使用模板 - 动态注入变量
import { ChatPromptTemplate } from '@langchain/core/prompts';

const template = ChatPromptTemplate.fromMessages([
  ['system', '你是一个 {role} 专家。'],
  ['human', '{question}'],
]);

// 复用模板，只改变量
const prompt1 = await template.format({
  role: 'React',
  question: '如何使用 useState？',
});

const prompt2 = await template.format({
  role: 'Vue',
  question: '如何使用 ref？',
});
```

**前端类比**：

```typescript
// React 组件模板
function Greeting({ name, role }: Props) {
  return <div>Hello, {name}! You are a {role}.</div>;
}

// LangChain Prompt 模板
const template = ChatPromptTemplate.fromTemplate(
  'Hello, {name}! You are a {role}.'
);
```

## 🛠️ 基础用法

### 1. 简单字符串模板

```typescript
import { PromptTemplate } from '@langchain/core/prompts';

// 创建模板
const simpleTemplate = PromptTemplate.fromTemplate('用{style}风格解释：{topic}');

// 格式化
const prompt = await simpleTemplate.format({
  style: '幽默',
  topic: 'React Hooks',
});

console.log(prompt);
// 输出："用幽默风格解释：React Hooks"
```

### 2. 聊天消息模板

```typescript
import { ChatPromptTemplate } from '@langchain/core/prompts';

const chatTemplate = ChatPromptTemplate.fromMessages([
  ['system', '你是一个{role}助手，回答要{style}。'],
  ['human', '{question}'],
]);

// 格式化为消息列表
const messages = await chatTemplate.formatMessages({
  role: '前端开发',
  style: '简洁专业',
  question: '解释闭包',
});

// 直接传给 LLM
const response = await llm.invoke(messages);
```

## 🎨 进阶用法

### 技巧 1：MessagesPlaceholder - 插入历史对话

**前端类比**：类似 React 的 `children` 属性

```typescript
import { ChatPromptTemplate, MessagesPlaceholder } from '@langchain/core/prompts';
import { HumanMessage, AIMessage } from '@langchain/core/messages';

// 创建模板（包含历史对话占位符）
const templateWithHistory = ChatPromptTemplate.fromMessages([
  ['system', '你是一个有记忆的助手。'],
  new MessagesPlaceholder('chat_history'), // 👈 历史对话插入点
  ['human', '{input}'],
]);

// 使用
const history = [new HumanMessage('我叫小明'), new AIMessage('你好，小明！')];

const messages = await templateWithHistory.formatMessages({
  chat_history: history, // 👈 插入历史
  input: '我叫什么名字？',
});

const response = await llm.invoke(messages);
console.log(response.content); // "你叫小明"
```

**React 类比**：

```tsx
// React 的 children
function Layout({ children }: { children: React.ReactNode }) {
  return (
    <div>
      <Header />
      {children} {/* 插入任意内容 */}
      <Footer />
    </div>
  );
}

// LangChain 的 MessagesPlaceholder
const template = ChatPromptTemplate.fromMessages([
  ['system', 'You are an assistant'],
  new MessagesPlaceholder('history'), // 插入任意消息
  ['human', '{input}'],
]);
```

### 技巧 2：条件模板

```typescript
type UserType = 'beginner' | 'expert';

function createPromptForUser(userType: UserType) {
  const systemMessages: Record<UserType, string> = {
    beginner: '你是一个耐心的老师，用简单的语言解释，多举例子。',
    expert: '你是技术同行，直接给出技术细节和最佳实践。',
  };

  return ChatPromptTemplate.fromMessages([
    ['system', systemMessages[userType]],
    ['human', '{question}'],
  ]);
}

// 使用
const beginnerTemplate = createPromptForUser('beginner');
const expertTemplate = createPromptForUser('expert');
```

### 技巧 3：Few-Shot 示例（提供示例）

```typescript
/**
 * Few-Shot Learning：给 AI 提供示例，让它模仿
 * 类似"举例说明"
 */

const fewShotTemplate = ChatPromptTemplate.fromMessages([
  ['system', '你是一个代码生成助手。参考示例格式输出。'],

  // 示例 1
  ['human', '生成一个按钮组件'],
  [
    'ai',
    `\`\`\`typescript
function Button({ text }: { text: string }) {
  return <button>{text}</button>;
}
\`\`\``,
  ],

  // 示例 2
  ['human', '生成一个输入框组件'],
  [
    'ai',
    `\`\`\`typescript
function Input({ value, onChange }: InputProps) {
  return <input value={value} onChange={onChange} />;
}
\`\`\``,
  ],

  // 用户的真实问题
  ['human', '{user_request}'],
]);

// 使用
const messages = await fewShotTemplate.formatMessages({
  user_request: '生成一个计数器组件',
});

const response = await llm.invoke(messages);
console.log(response.content);
// AI 会模仿上面的格式输出计数器组件
```

## 📊 完整项目：代码生成助手

创建 `src/06-code-generator.ts`：

```typescript
import { ChatOpenAI } from '@langchain/openai';
import { ChatPromptTemplate } from '@langchain/core/prompts';
import 'dotenv/config';

/**
 * 代码生成助手
 * 功能：根据描述生成 React/Vue/Angular 代码
 */

type Framework = 'react' | 'vue' | 'angular';

class CodeGenerator {
  private llm: ChatOpenAI;
  private templates: Record<Framework, ChatPromptTemplate>;

  constructor() {
    this.llm = new ChatOpenAI({
      modelName: 'gpt-4o-mini',
      temperature: 0.3, // 代码生成要更确定性
    });

    this.templates = {
      react: ChatPromptTemplate.fromMessages([
        [
          'system',
          `你是 React 代码生成专家。
规则：
1. 使用 TypeScript
2. 使用函数组件和 Hooks
3. 代码要有注释
4. 包含类型定义

示例格式：
\`\`\`typescript
interface ButtonProps {{
  text: string;
  onClick: () => void;
}}

function Button({{ text, onClick }}: ButtonProps) {{
  return <button onClick={{onClick}}>{{text}}</button>;
}}
\`\`\``,
        ],
        ['human', '生成组件：{description}'],
      ]),

      vue: ChatPromptTemplate.fromMessages([
        [
          'system',
          `你是 Vue 3 代码生成专家。
规则：
1. 使用 <script setup> 语法
2. 使用 TypeScript
3. 使用 Composition API
4. 代码要有注释`,
        ],
        ['human', '生成组件：{description}'],
      ]),

      angular: ChatPromptTemplate.fromMessages([
        [
          'system',
          `你是 Angular 代码生成专家。
规则：
1. 使用最新的 Angular 语法
2. 使用 TypeScript
3. 包含 @Component 装饰器
4. 代码要有注释`,
        ],
        ['human', '生成组件：{description}'],
      ]),
    };
  }

  async generate(framework: Framework, description: string): Promise<string> {
    const template = this.templates[framework];
    const messages = await template.formatMessages({ description });
    const response = await this.llm.invoke(messages);
    return response.content as string;
  }
}

// 使用示例
async function main() {
  const generator = new CodeGenerator();

  console.log('🔵 React 组件：');
  console.log('='.repeat(50));
  const reactCode = await generator.generate('react', '一个带加载状态的按钮，点击后显示 loading');
  console.log(reactCode);

  console.log('\n🟢 Vue 3 组件：');
  console.log('='.repeat(50));
  const vueCode = await generator.generate('vue', '一个带验证的输入框，实时显示错误提示');
  console.log(vueCode);
}

main();
```

## 🎯 实战：带记忆的聊天助手

创建 `src/07-chat-with-memory.ts`：

```typescript
import { ChatOpenAI } from '@langchain/openai';
import { ChatPromptTemplate, MessagesPlaceholder } from '@langchain/core/prompts';
import { HumanMessage, AIMessage, BaseMessage } from '@langchain/core/messages';
import * as readline from 'readline';
import 'dotenv/config';

/**
 * 带记忆的聊天助手
 * 演示 MessagesPlaceholder 的使用
 */

class ChatAssistant {
  private llm: ChatOpenAI;
  private template: ChatPromptTemplate;
  private history: BaseMessage[] = [];

  constructor() {
    this.llm = new ChatOpenAI({ modelName: 'gpt-4o-mini' });

    // 创建模板（包含历史对话）
    this.template = ChatPromptTemplate.fromMessages([
      ['system', '你是一个友好的助手，记住之前的对话内容。'],
      new MessagesPlaceholder('history'), // 👈 历史对话占位符
      ['human', '{input}'],
    ]);
  }

  async chat(input: string): Promise<string> {
    // 格式化消息
    const messages = await this.template.formatMessages({
      history: this.history,
      input,
    });

    // 调用 LLM
    const response = await this.llm.invoke(messages);
    const aiResponse = response.content as string;

    // 更新历史（只保存用户输入和 AI 回复，不包含 system）
    this.history.push(new HumanMessage(input));
    this.history.push(new AIMessage(aiResponse));

    // 限制历史长度（保留最近 10 轮）
    if (this.history.length > 20) {
      this.history = this.history.slice(-20);
    }

    return aiResponse;
  }

  clearHistory() {
    this.history = [];
  }
}

// 测试
async function main() {
  const assistant = new ChatAssistant();

  console.log(await assistant.chat('我叫小明'));
  // 输出："你好，小明！很高兴认识你。"

  console.log(await assistant.chat('我今年25岁'));
  // 输出："好的，小明。你今年25岁。"

  console.log(await assistant.chat('我叫什么名字，今年多大？'));
  // 输出："你叫小明，今年25岁。"
}

main();
```

## 💡 常见问题 FAQ

**Q: Prompt Template 和直接拼字符串有什么区别？**
A:

- ✅ 模板：可复用、易维护、类型安全
- ❌ 拼字符串：一次性、难维护、易出错

**Q: MessagesPlaceholder 什么时候用？**
A: 当你需要在固定的 System/Human 消息之间插入动态的对话历史时。

**Q: Few-Shot 示例多少个合适？**
A: 一般 2-5 个示例最佳。太多会浪费 token，太少效果不明显。

**Q: 如何保存模板到文件？**
A: 可以用 JSON 格式保存：

```typescript
const templateData = {
  system: '你是 {role}',
  human: '{question}',
};

// 加载时重建模板
const template = ChatPromptTemplate.fromMessages([
  ['system', templateData.system],
  ['human', templateData.human],
]);
```
