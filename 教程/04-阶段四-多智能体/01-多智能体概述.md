# 01 - 多智能体概述

> **核心理念**：多个专业 Agent 协作，完成复杂任务

## 🎯 学习目标

- 理解为什么需要多智能体
- 掌握两种协作模式
- 学会设计 Agent 团队
- 理解角色分工

---

## 📝 为什么需要多智能体？

### 单Agent的局限

```typescript
// ❌ 单个 Agent 处理所有事情
用户："写一篇关于 AI 的文章，然后翻译成英文，最后生成 PPT"

单 Agent：
- 要懂写作 ✍️
- 要懂翻译 🌍
- 要懂 PPT 设计 🎨
- 工具太多，容易混乱
```

### 多 Agent 协作

```typescript
// ✅ 多个专业 Agent 协作
Writer Agent:  写文章 ✍️
Translator Agent: 翻译 🌍
Designer Agent: 生成 PPT 🎨

每个 Agent 只做自己擅长的事！
```

---

## 🛠️ 两种协作模式

### 1. 主管-员工模式（Supervisor Pattern）

**结构**：

```
        ┌──────────────┐
        │  Supervisor  │ (主管)
        │  (路由决策)  │
        └──────┬───────┘
               │
       ┌───────┼───────┐
       ↓       ↓       ↓
   ┌──────┐ ┌──────┐ ┌──────┐
   │Worker│ │Worker│ │Worker│ (员工)
   │  A   │ │  B   │ │  C   │
   └──────┘ └──────┘ └──────┘
```

**特点**：
- 主管负责分配任务
- 员工各司其职
- 集中控制

---

### 2. 网状对等模式（Network Pattern）

**结构**：

```
   ┌─────┐
   │  A  │ ←→ ┌─────┐
   └──┬──┘    │  B  │
      ↓       └──┬──┘
   ┌─────┐      ↓
   │  C  │ ←→ ┌─────┐
   └─────┘    │  D  │
              └─────┘
```

**特点**：
- Agent 之间平等对话
- 自主决策
- 灵活协作

## 📊 示例：研究助手团队

创建 `src/24-research-team.ts`：

```typescript
import { Annotation, StateGraph } from '@langchain/langgraph';
import { ChatOpenAI } from '@langchain/openai';
import { HumanMessage, AIMessage, BaseMessage } from '@langchain/core/messages';
import 'dotenv/config';

/**
 * 研究助手团队（主管-员工模式）
 * - Supervisor: 分配任务
 * - Researcher: 搜索资料
 * - Writer: 撰写报告
 * - Reviewer: 审查内容
 */

// State 定义
const TeamState = Annotation.Root({
  messages: Annotation<BaseMessage[]>({
    reducer: (x, y) => x.concat(y),
    default: () => [],
  }),
  currentAgent: Annotation<string>({
    reducer: (_, y) => y,
    default: () => 'supervisor',
  }),
  researchData: Annotation<string>({
    reducer: (_, y) => y,
    default: () => '',
  }),
  report: Annotation<string>({
    reducer: (_, y) => y,
    default: () => '',
  }),
});

type State = typeof TeamState.State;

const llm = new ChatOpenAI({ modelName: 'gpt-4o-mini', temperature: 0 });

// 主管节点：分配任务
async function supervisorNode(state: State): Promise<Partial<State>> {
  console.log('\n👔 主管: 分析任务...');

  // 简单的任务路由逻辑
  if (!state.researchData) {
    console.log('   → 分配给 Researcher');
    return { currentAgent: 'researcher' };
  }

  if (!state.report) {
    console.log('   → 分配给 Writer');
    return { currentAgent: 'writer' };
  }

  console.log('   → 分配给 Reviewer');
  return { currentAgent: 'reviewer' };
}

// 研究员节点：搜索资料
async function researcherNode(state: State): Promise<Partial<State>> {
  console.log('\n🔍 Researcher: 搜索资料...');

  const userMsg = state.messages[0];
  const topic = userMsg.content as string;

  // 模拟搜索
  const data = `关于"${topic}"的研究资料：
1. 定义和背景
2. 主要特点
3. 应用场景`;

  console.log('   ✅ 资料已收集');

  return {
    researchData: data,
    currentAgent: 'supervisor',
  };
}

// 作家节点：撰写报告
async function writerNode(state: State): Promise<Partial<State>> {
  console.log('\n✍️ Writer: 撰写报告...');

  const report = `# 研究报告

${state.researchData}

## 结论
基于以上研究，我们得出...`;

  console.log('   ✅ 报告已完成');

  return {
    report,
    currentAgent: 'supervisor',
  };
}

// 审查员节点：审查内容
async function reviewerNode(state: State): Promise<Partial<State>> {
  console.log('\n📋 Reviewer: 审查报告...');

  // 简单审查
  const feedback = '报告质量良好，建议发布。';

  console.log('   ✅', feedback);

  return {
    currentAgent: 'finish',
    messages: [new AIMessage(`报告已完成并通过审查:\n\n${state.report}`)],
  };
}

// 路由函数
function router(state: State): string {
  return state.currentAgent;
}

// 构建图
const graph = new StateGraph(TeamState)
  .addNode('supervisor', supervisorNode)
  .addNode('researcher', researcherNode)
  .addNode('writer', writerNode)
  .addNode('reviewer', reviewerNode)
  .addNode('finish', () => ({}))

  .addEdge('__start__', 'supervisor')

  .addConditionalEdges('supervisor', router, {
    researcher: 'researcher',
    writer: 'writer',
    reviewer: 'reviewer',
  })

  .addEdge('researcher', 'supervisor')
  .addEdge('writer', 'supervisor')
  .addEdge('reviewer', 'finish')
  .addEdge('finish', '__end__')

  .compile();

// 测试
async function testTeam() {
  console.log('🏢 启动研究团队');
  console.log('='.repeat(60));

  const result = await graph.invoke({
    messages: [new HumanMessage('研究 AI Agent 的应用')],
  });

  console.log('\n' + '='.repeat(60));
  console.log('📊 最终结果:');
  console.log(result.report);
}

testTeam();
```

**运行结果**：

```
🏢 启动研究团队
============================================================

👔 主管: 分析任务...
   → 分配给 Researcher

🔍 Researcher: 搜索资料...
   ✅ 资料已收集

👔 主管: 分析任务...
   → 分配给 Writer

✍️ Writer: 撰写报告...
   ✅ 报告已完成

👔 主管: 分析任务...
   → 分配给 Reviewer

📋 Reviewer: 审查报告...
   ✅ 报告质量良好，建议发布。

============================================================
📊 最终结果:
# 研究报告

关于"研究 AI Agent 的应用"的研究资料：
1. 定义和背景
2. 主要特点
3. 应用场景

## 结论
基于以上研究，我们得出...
```
