# 03 - 节点与边

> **核心理念**：节点是处理单元，边是流转路径，组合起来就是 Agent 的大脑

## 🎯 学习目标

- 掌握如何定义节点（Node）
- 理解边（Edge）的类型
- 学会使用条件边（Conditional Edge）
- 构建第一个完整的图（Graph）

---

## 📝 核心概念

### 节点（Node）是什么？

**前端类比**：

```typescript
// React 组件
function Button({ onClick }: Props) {
  return <button onClick={onClick}>Click</button>;
}

// LangGraph 节点
function thinkNode(state: State): Partial<State> {
  // 处理逻辑
  return { currentStep: 'action' };
}
```

**节点的职责**：

1. 接收当前 State
2. 执行业务逻辑
3. 返回新的 State（部分更新）

---

### 边（Edge）是什么？

**前端类比**：

```typescript
// React Router
<Routes>
  <Route path="/" element={<Home />} />
  <Route path="/about" element={<About />} />
</Routes>

// LangGraph 边
graph
  .addEdge('home', 'about')        // 固定跳转
  .addConditionalEdges('home', router);  // 条件跳转
```

**边的类型**：

1. **普通边**：A → B（固定流转）
2. **条件边**：A → B 或 C（根据条件决定）
3. **起始边**：START → A
4. **结束边**：A → END

## 🛠️ 定义节点

### 1. 最简单的节点

```typescript
import { Annotation } from '@langchain/langgraph';
import { BaseMessage } from '@langchain/core/messages';

// State 定义
const State = Annotation.Root({
  messages: Annotation<BaseMessage[]>({
    reducer: (x, y) => x.concat(y),
  }),
  counter: Annotation<number>({
    reducer: (x, y) => x + y,
    default: () => 0,
  }),
});

type StateType = typeof State.State;

// 节点函数
function incrementNode(state: StateType): Partial<StateType> {
  console.log('当前计数:', state.counter);
  return { counter: 1 }; // 返回部分 State，会被 reducer 处理
}

function logNode(state: StateType): Partial<StateType> {
  console.log('最终计数:', state.counter);
  return {}; // 不更新 State
}
```

---

### 2. 异步节点

```typescript
import { ChatOpenAI } from '@langchain/openai';
import { HumanMessage, AIMessage } from '@langchain/core/messages';

const llm = new ChatOpenAI({ modelName: 'gpt-4o-mini' });

// 异步节点（支持 async/await）
async function chatNode(state: StateType): Promise<Partial<StateType>> {
  const lastMessage = state.messages[state.messages.length - 1];

  // 调用 LLM
  const response = await llm.invoke([lastMessage]);

  // 返回新消息
  return {
    messages: [new AIMessage(response.content as string)],
  };
}
```

### 3. 带工具的节点

```typescript
import { ToolMessage } from '@langchain/core/messages';

// 工具执行节点
async function toolNode(state: StateType): Promise<Partial<StateType>> {
  const lastMessage = state.messages[state.messages.length - 1];

  // 检查是否有工具调用
  if ('tool_calls' in lastMessage && lastMessage.tool_calls) {
    const toolMessages: ToolMessage[] = [];

    // 执行所有工具
    for (const toolCall of lastMessage.tool_calls) {
      const result = await executeTool(toolCall.name, toolCall.args);

      toolMessages.push(
        new ToolMessage({
          content: result,
          tool_call_id: toolCall.id!,
        })
      );
    }

    return { messages: toolMessages };
  }

  return {};
}
```

## 🎨 定义边

### 1. 普通边（固定流转）

```typescript
import { StateGraph } from '@langchain/langgraph';

const graph = new StateGraph(State)
  .addNode('node_a', nodeA)
  .addNode('node_b', nodeB)
  .addNode('node_c', nodeC)

  // 固定流转：A → B → C
  .addEdge('__start__', 'node_a') // START → A
  .addEdge('node_a', 'node_b') // A → B
  .addEdge('node_b', 'node_c') // B → C
  .addEdge('node_c', '__end__') // C → END

  .compile();
```

**可视化**：

```
START → node_a → node_b → node_c → END
```

---

### 2. 条件边（动态流转）

```typescript
// 路由函数：根据 State 决定下一个节点
function router(state: StateType): string {
  if (state.counter < 10) {
    return 'increment'; // 继续增加
  } else {
    return 'finish'; // 结束
  }
}

const graph = new StateGraph(State)
  .addNode('increment', incrementNode)
  .addNode('finish', finishNode)

  // 条件边：根据 router 函数决定走哪条路
  .addConditionalEdges('increment', router, {
    increment: 'increment', // 循环
    finish: 'finish', // 结束
  })

  .addEdge('__start__', 'increment')
  .addEdge('finish', '__end__')

  .compile();
```

**可视化**：

```
       ┌─────────────┐
START → │  increment  │
       └──────┬──────┘
              ↓
          router()
         ╱        ╲
    < 10 ╱          ╲ >= 10
        ╱            ╲
       ↓              ↓
  [循环回去]      [finish] → END
```

## 📊 完整示例：计数器 Agent

创建 `src/13-counter-agent.ts`：

```typescript
import { Annotation, StateGraph } from '@langchain/langgraph';
import 'dotenv/config';

/**
 * 计数器 Agent
 * 功能：从 0 计数到 10
 */

// 1. 定义 State
const CounterState = Annotation.Root({
  counter: Annotation<number>({
    reducer: (x, y) => x + y,
    default: () => 0,
  }),
  history: Annotation<string[]>({
    reducer: (x, y) => x.concat(y),
    default: () => [],
  }),
});

type State = typeof CounterState.State;

// 2. 定义节点
function incrementNode(state: State): Partial<State> {
  const newValue = state.counter + 1;
  console.log(`📊 计数: ${state.counter} → ${newValue}`);

  return {
    counter: 1, // 加 1（会被 reducer 处理）
    history: [`计数到 ${newValue}`],
  };
}

function finishNode(state: State): Partial<State> {
  console.log(`✅ 完成！最终值: ${state.counter}`);
  console.log(`📋 历史记录:`, state.history);
  return {};
}

// 3. 定义路由函数
function shouldContinue(state: State): string {
  if (state.counter < 10) {
    return 'increment';
  }
  return 'finish';
}

// 4. 构建图
const graph = new StateGraph(CounterState)
  .addNode('increment', incrementNode)
  .addNode('finish', finishNode)

  // 起始边
  .addEdge('__start__', 'increment')

  // 条件边（关键！）
  .addConditionalEdges('increment', shouldContinue, {
    increment: 'increment', // 继续计数
    finish: 'finish', // 结束
  })

  // 结束边
  .addEdge('finish', '__end__')

  .compile();

// 5. 运行
async function main() {
  console.log('🚀 启动计数器 Agent\n');

  const result = await graph.invoke({});

  console.log('\n最终状态:', result);
}

main();
```

**运行结果**：

```
🚀 启动计数器 Agent

📊 计数: 0 → 1
📊 计数: 1 → 2
📊 计数: 2 → 3
📊 计数: 3 → 4
📊 计数: 4 → 5
📊 计数: 5 → 6
📊 计数: 6 → 7
📊 计数: 7 → 8
📊 计数: 8 → 9
📊 计数: 9 → 10
✅ 完成！最终值: 10
📋 历史记录: [
  '计数到 1', '计数到 2', '计数到 3', '计数到 4', '计数到 5',
  '计数到 6', '计数到 7', '计数到 8', '计数到 9', '计数到 10'
]

最终状态: { counter: 10, history: [...] }
```

## 🎯 实战：猜数字游戏

创建 `src/14-guess-number.ts`：

```typescript
import { Annotation, StateGraph } from '@langchain/langgraph';
import { ChatOpenAI } from '@langchain/openai';
import { HumanMessage, AIMessage, BaseMessage } from '@langchain/core/messages';

/**
 * 猜数字游戏
 * AI 猜一个 1-100 的数字，最多 7 次机会
 */

// State 定义
const GameState = Annotation.Root({
  messages: Annotation<BaseMessage[]>({
    reducer: (x, y) => x.concat(y),
    default: () => [],
  }),
  targetNumber: Annotation<number>({
    reducer: (_, y) => y,
    default: () => Math.floor(Math.random() * 100) + 1,
  }),
  guessCount: Annotation<number>({
    reducer: (x, y) => x + y,
    default: () => 0,
  }),
  lastGuess: Annotation<number | null>({
    reducer: (_, y) => y,
    default: () => null,
  }),
  status: Annotation<'playing' | 'won' | 'lost'>({
    reducer: (_, y) => y,
    default: () => 'playing',
  }),
});

type State = typeof GameState.State;

const llm = new ChatOpenAI({ modelName: 'gpt-4o-mini', temperature: 0.7 });

// 节点1：AI 猜数字
async function guessNode(state: State): Promise<Partial<State>> {
  const prompt =
    state.messages.length === 0
      ? '我想让你猜一个1到100之间的数字。请给出你的第一次猜测（只回复数字）。'
      : `${state.messages[state.messages.length - 1].content}。请继续猜测（只回复数字）。`;

  const response = await llm.invoke([new HumanMessage(prompt)]);
  const guess = parseInt((response.content as string).match(/\d+/)?.[0] || '50');

  console.log(`🤔 AI 猜测: ${guess}`);

  return {
    lastGuess: guess,
    guessCount: 1,
    messages: [new AIMessage(`我猜 ${guess}`)],
  };
}

// 节点2：检查结果
function checkNode(state: State): Partial<State> {
  const { lastGuess, targetNumber, guessCount } = state;

  if (lastGuess === targetNumber) {
    console.log(`🎉 猜对了！用了 ${guessCount} 次`);
    return {
      status: 'won',
      messages: [new HumanMessage(`正确！答案是 ${targetNumber}`)],
    };
  }

  if (guessCount >= 7) {
    console.log(`😢 失败！答案是 ${targetNumber}`);
    return {
      status: 'lost',
      messages: [new HumanMessage(`游戏结束！答案是 ${targetNumber}`)],
    };
  }

  const hint = lastGuess! < targetNumber ? '太小了' : '太大了';
  console.log(`💡 提示: ${hint}`);

  return {
    messages: [new HumanMessage(hint)],
  };
}

// 路由函数
function router(state: State): string {
  if (state.status === 'won' || state.status === 'lost') {
    return 'end';
  }
  return 'guess';
}

// 构建图
const graph = new StateGraph(GameState)
  .addNode('guess', guessNode)
  .addNode('check', checkNode)
  .addNode('end', () => ({}))

  .addEdge('__start__', 'guess')
  .addEdge('guess', 'check')

  .addConditionalEdges('check', router, {
    guess: 'guess', // 继续猜
    end: 'end', // 结束
  })

  .addEdge('end', '__end__')

  .compile();

// 运行
async function main() {
  console.log('🎮 开始猜数字游戏\n');

  const result = await graph.invoke({});

  console.log('\n游戏结果:', result.status);
  console.log('目标数字:', result.targetNumber);
  console.log('猜测次数:', result.guessCount);
}

main();
```

## 🔍 高级技巧

### 1. 多条件路由

```typescript
function complexRouter(state: State): string {
  if (state.error) {
    return 'handle_error';
  }
  if (state.needConfirmation) {
    return 'wait_for_user';
  }
  if (state.taskComplete) {
    return 'finish';
  }
  return 'continue';
}

graph.addConditionalEdges('main', complexRouter, {
  handle_error: 'error_node',
  wait_for_user: 'confirmation_node',
  finish: 'finish_node',
  continue: 'main', // 循环
});
```

### 2. 并行执行

```typescript
// 虽然 LangGraph 是串行的，但可以在节点内并行执行
async function parallelNode(state: State): Promise<Partial<State>> {
  // 同时执行多个任务
  const [result1, result2, result3] = await Promise.all([task1(), task2(), task3()]);

  return { results: [result1, result2, result3] };
}
```

### 3. 动态添加节点

```typescript
// 根据配置动态构建图
function buildGraph(tools: Tool[]) {
  const graph = new StateGraph(State);

  // 为每个工具创建节点
  tools.forEach(tool => {
    graph.addNode(tool.name, async state => {
      const result = await tool.execute(state);
      return { toolResults: { [tool.name]: result } };
    });
  });

  return graph.compile();
}
```

## 📚 知识点总结

### ✅ 你已经掌握

1. **节点定义**
   - 同步/异步节点
   - 返回部分 State
   - 节点可以不返回任何更新

2. **边的类型**
   - 普通边：固定流转
   - 条件边：动态决策
   - 起始边/结束边

3. **条件路由**
   - 路由函数返回字符串
   - 根据 State 决定路径
   - 支持循环

4. **图的构建**
   - `addNode`：添加节点
   - `addEdge`：添加普通边
   - `addConditionalEdges`：添加条件边
   - `compile`：编译图

| LangGraph 概念 | 前端类比    | 说明         |
| -------------- | ----------- | ------------ |
| 节点           | React 组件  | 处理单元     |

| 条件边         | 路由守卫    | if/else 跳转 |
| 路由函数       | useNavigate | 决定跳转     |
| 图             | 路由配置    | 整体流程     |

## 💡 常见问题 FAQ

**Q: 节点可以直接修改 State 吗？**
A: 不可以！节点只能返回新的 State，由 reducer 处理合并。

**Q: 条件边的路由函数可以是异步的吗？**
A: 不可以。路由函数必须是同步的，返回字符串。

**Q: 可以从任意节点跳到 END 吗？**
A: 可以，但建议先跳到一个"清理节点"，再跳到 END。

**Q: 如何调试图的执行流程？**
A: 使用 `graph.stream()` 查看每一步：

```typescript
for await (const step of graph.stream(input)) {
  console.log('当前步骤:', step);
}
```
